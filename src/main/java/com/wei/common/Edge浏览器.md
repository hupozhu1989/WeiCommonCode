### 阅读模式

地址栏开头加 **read:**   或  左键选中内容然后右键**在沉浸式阅读器中打开所选内容**

朗读此页内容

### PDF阅读器

绘制 和 突出显示 功能很好用

## 玩转标签页

### 垂直标签页

标签页上右键**打开垂直标签页**，窗口顶部右键**隐藏标题栏**，获得更大阅读视野

### 组管理

标签页上右键**将标签页添加到新组**

### 睡眠模式

在指定一段时间后，将非活动标签页置于睡眠模式
